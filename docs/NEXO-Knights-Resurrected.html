NEXO Knights: Resurrected

"NEXO Knights: Resurrected" is a "LEGO: NEXO Knights" fan-fiction written by Troy Carpenter. It takes place after the events of the TV series and includes the main ensemble of characters. It will be an untold number of seasons (books) long with six to twelve episodes (chapters) each season. It is written by Troy Augustus Carpenter
